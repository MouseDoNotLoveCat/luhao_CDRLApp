# 导入流程问题修复 - 测试指南

## 🧪 测试步骤

### 1. 启动应用
```bash
./start-dev.sh
```

### 2. 打开浏览器
- 前端：http://localhost:3000
- 进入导入页面

### 3. 测试修复后的流程

#### 步骤 1：上传文档
1. 选择 Word 文档
2. 点击"开始识别"
3. 等待识别完成

#### 步骤 2：进入编辑界面
1. 进入通知书预览界面
2. 点击"问题列表"按钮
3. 进入问题编辑界面

#### 步骤 3：验证选择框 ✅
- **预期**：表格左侧显示复选框
- **验证**：
  - 看到"选择框"列
  - 可以点击复选框选择问题

#### 步骤 4：选择问题 ✅
1. 选择至少 2 个问题
2. **验证**：
  - 统计信息中"已选择"数字增加
  - 表格中选中的行高亮显示

#### 步骤 5：修改问题
1. 修改选中问题的字段
2. 修改问题类别、严重程度等
3. **验证**：
  - 统计信息中"已修改"数字增加

#### 步骤 6：保存修改 ✅
1. 点击"保存修改"按钮
2. **预期**：
  - 如果没有选择问题，显示警告"请先选择要导入的问题"
  - 如果选择了问题，显示"修改已保存"
  - 进入确认界面

#### 步骤 7：验证确认界面 ✅
- **预期**：确认界面显示所有选中的问题
- **验证**：
  - 看到"选中的问题列表（共 N 个）"
  - 表格显示所有选中的问题
  - 显示所有问题字段

#### 步骤 8：完成导入
1. 点击"确认导入"
2. 等待导入完成
3. 验证导入结果
