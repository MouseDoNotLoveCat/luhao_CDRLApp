# 问题清单表格 - 快速使用指南

## 🎯 新功能快速开始

### 功能 1：表格单元格直接编辑

#### 支持编辑的字段
- 检查单位
- 检查工点
- 整改责任单位
- 整改责任人
- 销号人员
- 销号状态

#### 使用步骤
1. 点击"编辑"按钮进入编辑模式
2. 点击要编辑的单元格
3. 输入新值
4. 按 **Enter** 保存或点击其他地方自动保存
5. 按 **Escape** 取消编辑

#### 快捷键
| 快捷键 | 功能 |
|--------|------|
| Enter | 保存编辑 |
| Escape | 取消编辑 |
| Tab | 移动到下一个单元格 |

---

### 功能 2：三层级联下拉列表

#### 级联结构
```
问题类别（一级）
  ├── 工程质量
  │   ├── 混凝土工程
  │   │   ├── 原材料
  │   │   ├── 模板及支架
  │   │   └── ...
  │   ├── 路基工程
  │   └── ...
  ├── 施工安全
  ├── 管理行为
  └── 其它
```

#### 使用步骤
1. 打开编辑对话框（点击问题行的"编辑"按钮）
2. 选择"问题类别"（一级分类）
3. 选择"问题子类1"（二级分类）
4. 选择"问题子类2"（三级分类）
5. 点击"保存"

#### 特点
- ✅ 自动级联
- ✅ 自动重置下级
- ✅ 防止数据不一致

## 🎓 使用场景

### 场景 1：快速修改检查单位
```
1. 进入问题库
2. 点击"编辑"
3. 点击"检查单位"单元格
4. 输入新单位名称
5. 按 Enter 保存
⏱️ 耗时：5 秒
```

### 场景 2：完整编辑问题信息
```
1. 进入问题库
2. 点击问题行的"编辑"按钮
3. 在对话框中修改所有字段
4. 点击"保存"
⏱️ 耗时：30 秒
```

### 场景 3：批量修改责任人
```
1. 进入问题库
2. 点击"编辑"
3. 逐个点击"整改责任人"单元格修改
4. 按 Enter 保存
⏱️ 耗时：10 秒/条
```

## 💡 技巧和建议

### 技巧 1：快速编辑多个字段
```
1. 点击第一个单元格编辑
2. 按 Enter 保存
3. 点击下一个单元格
4. 重复步骤 2-3
```

### 技巧 2：使用级联下拉列表
```
1. 打开编辑对话框
2. 选择一级分类
3. 二级分类自动更新
4. 选择二级分类
5. 三级分类自动更新
```

### 技巧 3：取消编辑
```
按 Escape 键立即取消编辑
不会保存任何更改
```

## ⚠️ 注意事项

1. **编辑模式**
   - 必须先点击"编辑"按钮进入编辑模式
   - 单元格才能被编辑

2. **数据保存**
   - 单元格编辑会立即保存到本地
   - 后续需要实现后端 API 保存

3. **级联下拉列表**
   - 改变一级分类会重置二级和三级
   - 改变二级分类会重置三级

4. **支持的字段**
   - 只有 6 个字段支持直接编辑
   - 其他字段需要使用编辑对话框

## 🔍 常见问题

**Q: 为什么单元格不能编辑？**
A: 需要先点击"编辑"按钮进入编辑模式。

**Q: 编辑后数据会保存到数据库吗？**
A: 当前版本保存到本地，后续会实现后端 API 保存。

**Q: 如何取消编辑？**
A: 按 Escape 键或点击其他地方。

**Q: 三级分类为什么没有显示？**
A: 需要先选择二级分类，三级分类才会显示。

**Q: 可以编辑哪些字段？**
A: 检查单位、检查工点、整改责任单位、整改责任人、销号人员、销号状态。
